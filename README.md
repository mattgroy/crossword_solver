Автор: Ратков Максим (ratkovm8@gmail.com)


## Новое в версии:
* улучшенный алгоритм поиска решений
* новые возможности (настройка вывода решения, ввод данных с консоли,
возможность указывать кодировку словаря)


## Описание
Данное приложение заполняет поле кроссворда методом "брутфорса"


## Требования
* Python версии не ниже 3.4


## Состав
* Консольная версия: 'crossword_solver_console.py'
* Модули: 'resources/'


## Консольная версия
Справка по запуску: './crossword_solver_console.py --help'

Пример запуска: './crossword_solver_console.py -e cp1251 -d -w sol.txt dict.txt field.txt


## Подробности реализации:
Модули, отвечающие за парсинг входных данных и брутфорс решений
расположены в пакете resources:
* resources/crossword.py: содержит класс Solver, занимающийся непосредственным решением кроссворда
* resources/parser.py: содержит класс Parser, занимающийся обработкой и распознаванием входных данных
* resources/ruwords.txt: словарь русских слов

Можно указывать кодировку словаря (--encoding)
Приложение поддерживает 2 вида вывода решений - классический и в виде нарисованного заполненного поля (--draw)

Вид входных данных (данных о поле):
*направление* (*координаты начала*) *длина слова*
Например:
v (1, 2) 5
означает: вертикальное слово, первая буква - в 1 строке 2 столбце (отсчет от 0), длина слова - 5
Направление задаётся буквами H или V
